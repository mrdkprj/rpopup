# RMenu
Rust context/popup menu for Windows.  
Supports dark/light theme and color/size configuration. 
- Colors
    - Text color
    - Background color
    - Border color
- Size
    - Menu top/bottom margin
    - Menu item top/bottom and left/right padding
    - Font size/weight
